# GreenLoop Admin Dashboard: User Manual

## 1. Introduction
The GreenLoop Admin Dashboard is a web-based application designed for ULB (Urban Local Body) administrators to monitor waste collection, manage HKS workers, and track municipal targets.

## 2. Getting Started
- Open the dashboard in a modern web browser (Chrome, Edge, Safari).
- Log in with your corporate email and secure password.
- Multi-factor authentication (MFA) will be required for administrative roles.

## 3. Dashboard Overview
The home screen provides four main Key Performance Indicators (KPIs):
1. **Registered Households**: Total households participating in the program.
2. **Current Month Tonnes**: Amount of waste collected in kg/tonnes.
3. **Citizen NPS**: Net Promoter Score based on resident feedback.
4. **HKS Activity**: Percentage of HKS members active today across all wards.

## 4. Ward Management
- Navigate to the **"Wards"** tab.
- Select a ward from the list to see it highlighted on the map.
- **Color Coding**: 
  - Green zones indicate successful collection today.
  - Red zones indicate delays or incomplete routes.
  - Grey zones indicate areas with pending registrations.

## 5. User & Role Management
- Navigate to **"User Management"**.
- Admins can add new HKS workers and assign them to specific wards.
- Residents are automatically added during onboarding but can be manually adjusted for household reassignment.

## 6. Real-time Monitoring
The **"Map View"** uses live GeoJSON data to track collection progress:
- **HKS Tracking**: View the real-time position of collection vehicles and HKS teams.
- **Incident Heatmap**: View areas with high complaint volumes or poor segregation accuracy.

## 7. Generating Progress Reports
- Go to the **"Reports"** section.
- Select your date range (e.g., last quarter).
- Click **"Generate Report"** to view on-screen analytics.
- Use the **"Export PDF"** button (top right) to download a professional municipal report for presentation.
